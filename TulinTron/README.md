## 🎮 使用指南 (全功能解析)

### 模式一：靓号与顺子模式 (Leopard/Straight)
自动寻找末尾 N 位相同或连续的地址：
```bash
# 生成末尾 8 位相同的豹子号 (例: T...88888888)
./tulinTron -l 8

# [v2.7 新增]生成末尾 8 位相同的伪豹子号 (例: T...AAAaaAAa)
./tulinTron -lf 8

# 生成末尾 8 位连续的顺子号 (例: T...12345678)
./tulinTron -s 8

# [v2.6 新增] 生成末尾 7 位纯数字顺子 (速度极快)
./tulinTron -sn 7
```

### 模式二：精准匹配模式 (Matching)
通过规则文件进行前后缀精准匹配，支持最多 32 个规则同时并行，完全不降速。最新v2.8匹配模式支持暴力加速，速度提升30%以上
```bash
# p.txt 格式: 前缀*后缀 (每行一个规则，最多32行)
# 例:
# TABC*XYZ
# T888*666
# T999*999
./tulinTron -m p.txt
```

### 模式三：随机纯数字模式 (Number)
生成末尾 N 位纯数字地址：
```bash
# 生成末尾 15 位纯数字地址
./tulinTron -n 15
```

### 模式四：多模同开 (终极榨干算力)
匹配 + 豹子 + 顺子 + 数字 多种模式同时运行，算力零损耗，一网打尽：
```bash
# 同时运行所有模式：32个匹配规则 + 8位豹子 + 9位顺子 + 15位纯数字
./tulinTron -m p.txt -l 8 -s 9 -n 15
```

### 🔒 高级特性：私钥加/解密模式
为了保障绝对的资产安全，计算出来的私钥支持加密存储落地，拒绝明文泄露：
```bash
# -k 参数: 设定加密秘钥 (配置后，计算结果会在内存中加密再写入本地文件)
# 加密计算模式 
./tulinTron -l 8 -k

# 启动后，提示您输入一段秘钥（密码）。
# 您自己想一段密码输进去后按回车就好
#（注意：输入密码时屏幕上不会显示任何字符星星，这是故意的防偷窥设计，闭着眼睛打完按回车就行），千万要记住这个密码！！！
# 之后扫到的靓号的私钥就会变成一串“天书”，没有密码谁也拿不走你的资产。

# --decrypt: 解密模式 (提取私钥时使用)
# -i 输入文件 | -o 输出文件
./tulinTron --decrypt -i result.txt

#（说明：result.txt 是扫出来的结果文件，根据您实际文件的名字修改，比如有时叫 6_N.txt）
# 按下回车后，系统会再次让您蒙眼输入刚才设置的密码。
# 如果密码输对了，软件就会在旁边变出一个 `decrypted.txt` 文件，里面就是您完整无缺的真私钥了！
# 如果密码错误，解密出来的私钥是乱码
```
